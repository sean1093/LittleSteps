# Public Assets

請將以下 PWA 圖標放置於此目錄：

- `pwa-192x192.png` - 192x192px PWA 圖標
- `pwa-512x512.png` - 512x512px PWA 圖標
- `apple-touch-icon.png` - 180x180px Apple Touch 圖標
- `favicon.ico` - Favicon
- `vite.svg` - Vite logo (暫時用)
- `og-image.png` - Open Graph 分享預覽圖（建議 1200x630px）

## 圖標設計建議

- 使用溫馨的粉嫩色系（#F472B6）
- 簡單的寶寶相關圖示（如腳印、奶瓶、愛心等）
- 圓角設計
- 背景色使用 #FAFAF9（暖白色）
